# Mission69 Nova V5.1

Correctif centré sur le tableau des poids.

## Changements
- Le tableau des poids contient maintenant un formulaire d’ajout intégré.
- Le bouton `+ ligne` ne crée plus de ligne préremplie/confuse.
- Si une pesée existe déjà à la même date, l’app propose de la remplacer.
- Validation plus robuste du poids.
- Corrections plus fiables dans `Tableaux → Poids`.
- Suppression confirmée toujours disponible.
- Migration automatique depuis Nova V5, Withings V4, Studio V3, Elite V2 et Dream V1.

## Installation
1. Dézipper.
2. Uploader tout le contenu à la racine du dépôt GitHub.
3. Commit conseillé : `Mission69 Nova V5.1`.
4. Attendre GitHub Pages.
5. Si l’ancienne version reste visible : Backup → Forcer mise à jour / vider cache.
